# Inventory Management Automation

## 🚀 What this project does
This project automates inventory tracking and updates using n8n workflows.

## ⚙️ Tech Used
- n8n
- Airtable / Database
- API integrations

## 🔄 Workflow
- Tracks product stock
- Updates quantity automatically
- Handles SKU-based inventory
- Sends alerts for low stock
